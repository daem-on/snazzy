Snazzy.

This game is an implementation of Cards Against Humanity, much like Pretend You're Xyzzy. Under development.

Design differences to PTYX: user interface was designed in a completely different way, with more emphasis on good UX and aesthetics. Responses are merged into calls when cards are revealed. The whole game can be easily self-hosted anywhere.

Creating a room

Rooms are automatically created from the direct links to games. Use snazzy.daemonw.com#room-name to create a room, which will also be the direct link for others.
